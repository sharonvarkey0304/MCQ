# Flutter MCQ App with Real-Time Scoring

This is a cross-platform mobile application built using **Flutter** and **Dart**. The app provides an interactive Multiple Choice Question (MCQ) experience where users can test their knowledge, receive instant feedback on pre-selected answers, and view their final results on a scoring board.



## Key Features
* **Interactive MCQ Engine:** A smooth, gesture-based interface for navigating through questions.
* **Pre-selected Answer Validation:** Answers are pre-configured in the backend logic, allowing the app to provide immediate "Correct" or "Incorrect" feedback.
* **Live Scoring:** Tracks user performance in real-time as they progress through the quiz.
* **Final Scoreboard:** A dedicated results screen that summarizes the total score, correct answers, and percentage achieved.
* **Cross-Platform:** Runs seamlessly on both Android and iOS from a single codebase.

## Project Structure
* `lib/models/`: Contains the Question and Answer data structures.
* `lib/screens/`: UI code for the Quiz page and the Scoring Board.
* `lib/data/`: The repository for the pre-selected questions and answers.
* `lib/widgets/`: Reusable UI components like custom buttons and progress bars.
